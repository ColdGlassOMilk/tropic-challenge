# :palm_tree: Tropic Dev Challenge

## :beach_umbrella: Prerequisites

| Library | Version |
| ------- | ------- |
| Ruby    | 3.2.1   |
| Rails   | 7.0.4   |

## :surfer: Development

1. Install dependencies & initialize the database:

   ```shell
   ./bin/setup
   ```

2. Start the development server:

   ```shell
   ./bin/dev
   ```

3. Navigate to [http://localhost:3000](http://localhost:3000)

## :shark: Testing

```shell
bundle exec rspec
```
